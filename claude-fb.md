### ⚠️ One discrepancy requiring your decision

**ADV Unit Overview format vs INT Focus format**

The two books handle unit-level learning goals differently:

- **INT:** `### Focus` → `"By the end of this unit, you should be able to:"` → `"- I can…"` statements (learner-facing, first person)
- **ADV:** `### Unit Overview` → `"By the end of the unit, learners will be able to:"` → third-person bullet list (e.g., `"identify sources of ambiguity"`)

The intermediate book is more aligned with the structural profiles' stated Can-Do standard (*"I can…"* format). The advanced book was developed earlier and uses the older third-person format throughout. The books are therefore internally consistent within themselves, but they use different formats for the same function.

This is not a failure in either book — both communicate the unit goals clearly — but it is a divergence in format that you may want to resolve before final publication. The question is whether you want both books to use `"I can…"` (which would require updating all 23 advanced unit overviews), or whether the difference is acceptable given the different learner-facing registers of the two books.
