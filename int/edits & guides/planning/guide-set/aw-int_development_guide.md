## 2. How To Use The Guide Set

### 2.1 For current project status

Start with:

- `aw-int_project_todo_list.md`

### 2.2 For planning or revising live content

Cross-check in this order:

1. `aw-int_unit_structural_profiles_revised.md`
2. `aw-int_activity_framework.md`
3. `aw-int_prompt-writing-policy.md`

### 2.3 For QA and post-QA review work

Use:

1. `aw-int_qa_checklist.md`
2. `aw-int_full-review_protocol.md`

These documents define the standards for their stages. Their findings and round artifacts should live outside this guide set.
