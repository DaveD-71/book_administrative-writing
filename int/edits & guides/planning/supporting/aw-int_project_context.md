## 1. What This Project Is

A workplace-writing textbook for lower-level administrative and professional learners.

**Title:** *Administrative Writing, Intermediate*  
**Target learners:** A2-B1+ workplace learners  
**Typical learner profiles:** administrative staff, junior officers, interns, and non-specialist professionals  
**Book identity:** simpler, more scaffolded, and more teachable than the advanced book while still preparing learners for routine workplace writing

## 3. Current Development Position

The intermediate track is earlier-stage than the advanced track.

Current priorities are:

1. establish the planning/control stack
2. run a baseline audit of the live unit files
3. build revised structural profiles
4. revise the book module by module against the planning stack

---

## 4. Quality Direction

The intermediate book should aim for:

- clear and teachable task progression
- concrete workplace realism
- strong support from explanation to guided writing to freer writing
- stable output expectations
- better whole-book control over repetition, scenario spread, and capstone logic
