## 2. Whole-Book Structural Pattern

The current live book already has a recognisable and reusable intermediate architecture.

Module distribution:

- Module 1: Units 1-3
- Module 2: Units 4-7
- Module 3: Units 8-11
- Module 4: Units 12-15
- Module 5: Units 16-18
- Module 6: Units 19-23

Dominant unit progression:

1. `Focus`
2. concept or scenario setup
3. one or more `Language` sections
4. one or more short `Practice` sections
5. `Guided Practice`
6. `Freer Practice`
7. `Extended Writing Task` in most units
8. `Reflection`
9. `Homework`

Book-level progression:

- Units 1-3 establish sentence, tone, and paragraph control
- Units 4-15 build standard workplace text types and email functions
- Units 16-18 shift into revision and editing
- Units 19-23 move into integrated, multi-audience, or portfolio-style work

This means the intermediate draft is not structurally random. It already has a usable progression model that can be formalised in the profile layer.

---

## 3. Output-Length Behavior

Current output control is present but not yet standardised at whole-book level.

Observed pattern:

- early and mid-book guided/freer tasks often use sentence counts such as `3-4`, `3-5`, `4-6`, `5-7`, or `6-8` sentences
- later integrated units still use sentence-count bands rather than a clearer word-range model
- the book uses labels such as `short email`, `paragraph`, `notice`, and `message`, but these are not yet normalised into a stable output framework across the whole book

Implication for planning:

- sentence counts are currently doing useful scaffolding work for A2-B1+ learners
- the structural profiles should record the current sentence-band model first, then decide later where stable word ranges are more appropriate
- late integrated units will need the clearest output-control treatment during profile drafting

---

## 4. Scenario And Task Families

The scenario base is appropriately ordinary and workplace-oriented for the book's level.

Strong current families:

- email basics
- clarification and requests
- delays, changes, and updates
- procedures and notices
- explanations and meeting summaries
- external inquiry and applicant/foreign-office communication
- editing for accuracy, clarity, and tone
- integrated multi-audience communication
- portfolio revision

Current whole-book shape:

- the book remains heavily email-centered
- report-style and paragraph-style writing are present, but usually as shorter bounded outputs
- Module 6 is the main point where the book becomes integrated and multi-document

Planning implication:

- the later integrated-task logic is already seeded in the live source
- the profile layer should preserve that progression rather than flattening the book into one repeated email-only pattern
