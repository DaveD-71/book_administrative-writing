# N10 Plan — Unit-by-Unit Language Instruction Strengthening
## *Administrative Writing, Advanced*

---

> **Purpose of this document**
> This file converts the `N10` framework into a concrete implementation plan for the existing `aw-adv_mod*_n9.md` files.
>
> It identifies, for every unit:
>
> - the likely `N10` intervention level
> - the main teaching gap to address
> - the exact section(s) where language-instruction strengthening should be added
> - the type of intervention to add there
>
> This is a planning document for execution. It does not yet change the module files.

---

## Intervention Key

- `A` = Conceptual framing
- `B` = Contrast explanation
- `C` = Worked example
- `D` = Model commentary / why-this-works explanation
- `E` = Transfer cue

---

## Module 1 — Foundations

### Unit 1 — Controlling Sentence Complexity

- `Level`: 3
- `Main gap`: the unit contains useful material, but the conceptual principle is still under-taught in places; key distinctions remain blurry.
- `Planned insertions`:
- `B1 Compare Two Versions`: add a short `D` note after the comparison prompts explaining that clarity depends on managing logical load, not simply making sentences shorter.
- `B2 Noticing Task`: revise/strengthen with a short `B` clarification that segmentation is a relationship across two sentences, not a property of one sentence.
- `C1 Coordination`: add `A` framing explaining when equal-weight ideas should stay coordinated rather than subordinated.
- `C2 Subordination`: add `B` explanation clarifying how subordinated contrast differs rhetorically from coordinated contrast.
- `C3 Clarity Patterns for Administrative Writing`: add `C` worked example for Pattern 3 and tighten the explanatory labels; define the arrow notation explicitly.
- `D2 Write Your Own Complex Sentence`: add `E` cue reminding learners to choose the pattern based on idea relationship, not sentence length.

### Unit 2 — Tone for Agency-Level Communication

- `Level`: 2
- `Main gap`: the unit teaches softening through examples and phrase lists, but under-explains why phrasing choices work and how audience affects register decisions.
- `Planned insertions`:
- `B1 Compare Two Versions`: add `D` explanation of how the revised version reduces blame, protects cooperation, and keeps the reader willing to respond.
- `C1 Softening Strategies`: add `A` framing explaining softening as a pragmatic tool, not indirectness for its own sake.
- `C2 Neutral Clarification Language`: add `B` explanation distinguishing neutral clarification from accusation or implied fault.
- `C3 Audience-Appropriate Tone`: add `C` worked mini-example showing how the same request shifts across internal, interagency, and foreign-office audiences.
- `D4 Tone Transformation`: add `E` cue linking audience choice to later freer writing.

### Unit 3 — Cohesion & Logical Flow

- `Level`: 2
- `Main gap`: cohesive devices and thematic progression are practised, but the unit needs clearer explanation of how coherence is built for a reader.
- `Planned insertions`:
- `B1 Compare Two Versions`: add `D` explanation of why the revised paragraph is easier to follow: clear topic movement, explicit sentence links, controlled reference.
- `C1 Cohesive Devices`: add `A` framing explaining that connectors signal relationships, not decoration.
- `C2 Reference and Continuity`: add `B` explanation of when repetition helps clarity and when reference improves flow.
- `C3 Thematic Progression`: add `C` worked example showing topic → support → implication movement in a short paragraph.
- `D5 Reorganise and Rewrite`: add `E` cue reminding learners to think in paragraph movement, not sentence list order.

## Likely First-Pass Priority Order

If `N10` is executed module by module, the likely priority order is:

1. Module 2
2. Module 3
3. Module 4
4. Module 5
5. Module 6
6. Module 1 final consistency sweep

Reason:

- Module 2 has a clear teaching gap around diplomatic phrasing and audience logic.
- Module 3 needs stronger conceptual treatment of cohesion, explanation, and summary logic.
- Modules 4–6 involve more complex rhetorical skills that require explanation, not only format.
- Module 1 already contains more explicit instructional material than much of the rest of the book, though Unit 1 still needs deep repair.
