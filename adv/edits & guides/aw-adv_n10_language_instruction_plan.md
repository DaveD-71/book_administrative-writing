# N10 Plan — Unit-by-Unit Language Instruction Strengthening
## *Administrative Writing, Advanced*

---

> **Purpose of this document**
> This file converts the `N10` framework into a concrete implementation plan for the existing `aw-adv_mod*_n9.md` files.
>
> It identifies, for every unit:
>
> - the likely `N10` intervention level
> - the main teaching gap to address
> - the exact section(s) where language-instruction strengthening should be added
> - the type of intervention to add there
>
> This is a planning and tracking document for execution. Update the review columns after each module is edited and then checked against the framework and the structural-profile goals.

---

## Intervention Key

- `A` = Conceptual framing
- `B` = Contrast explanation
- `C` = Worked example
- `D` = Model commentary / why-this-works explanation
- `E` = Transfer cue

---

## Alignment Requirement

For every unit in this plan, `N10` must confirm and, where necessary, make more visible the alignment between:

- the module goal or module purpose
- the unit-level learning goal / Can-Do statements
- the activity-level goals, especially across `B`, `C`, and `D`

So each unit entry below includes an `Alignment focus` line. This identifies the goal connection that the strengthened language instruction must help make explicit to the learner.

---

## Table Column Guide

Use the tables below as the tracking control for `N10`.

- `Activity`: the exact activity or sub-section where the intervention should be added
- `What`: the planned strengthening to be added there
- `Status`:
  - `Not started` = the planned insertion has not yet been made
  - `Added - review needed` = the insertion has been made, but not yet checked rigorously
  - `Added - cleanup needed` = something has been inserted, but the placement or wording is currently unsatisfactory
  - `Reviewed - pass` = the insertion has been added and checked satisfactorily
- `Framework matching`:
  - `Yes` = the current insertion matches the requirement type and expectations in `aw-adv_n10_language_instruction_framework.md`
  - `No` = the current insertion does not satisfactorily meet the framework requirement
  - `Not yet checked` = no rigorous framework check has been completed yet
- `Goals alignment`:
  - `Yes` = the current insertion clearly supports the relevant module purpose, unit goal, and activity goal in `aw-adv_unit_structural_profiles_revised.md`
  - `No` = the current insertion does not clearly align with those goals
  - `Not yet checked` = no rigorous goals-alignment check has been completed yet
