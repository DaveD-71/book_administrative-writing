## 1. What This Project Is

A **professional ELT textbook** for workplace learners, currently in the planning and editing phase. The book covers administrative and professional communication writing across 23 units organised into 5 modules.

**Title:** *Administrative Writing, Advanced*
**Series context:** This is the third level of a series. A Pre-Intermediate and an Intermediate volume also exist, though this project focuses on the Advanced level only.
**Target learners:** Primarily upper-intermediate (B1+/B2) workplace learners. The "Advanced" label is a deliberate commercial decision — the content is aspirational and carefully scaffolded to support learners reaching above their current level.
**Typical learner profiles:** Administrative officers, team leaders, analysts, junior-to-mid-level professionals in government, interagency, or international organisational contexts.
**Primary context:** The book's scenarios and model texts are oriented toward interagency and international administrative communication (e.g., IP offices, foreign ministries, coordinating agencies). This is the professional world the model texts inhabit.

## 3. Key Decisions Already Made

| Decision | Detail |
|---|---|
| **Level label** | "Advanced" on cover and throughout. Internal framing targets B1+/B2 learners in aspirational, scaffolded context. Do not change or contradict this. |
| **File format** | Markdown throughout. One file per module as working documents. |
| **Workflow** | Major content creation done collaboratively in Claude. Minor edits may be made independently between sessions. GitHub used for versioning. |
| **Planning-first approach** | Structure and activity variety are being planned before any unit-level editing begins. The unit structural profiles (planning table) must be completed before drafting starts. |
| **Group vs. self-study** | Dual-framing convention established once in a *How to Use This Book* section. Not duplicated in every activity instruction. "Discuss" activities are for class use; self-study learners are directed to write their response before continuing. |
| **Sentence counts → word counts** | All Freer Writing and Homework tasks currently use sentence-count targets. These will be replaced with word counts throughout. Progression: ~150w (Module 1) → ~200w (Modules 2–3) → ~250w (Modules 4–5) → ~300–350w (Module 6 / Units 22–23). |
| **Encoding** | The main unit file has been verified as clean UTF-8. Introduction files should be checked when next updated. |

## 7. Conventions & Style Notes

- **Spelling:** British English throughout (e.g., *organise*, *analyse*, *colour*, *practise* as verb)
- **Tone of instructions:** Direct and professional; second person ("Learners will…" in objectives; "Write…", "Identify…", "Discuss…" in activity instructions)
- **Model texts:** Every unit contains a weak original and an improved version. This is a core feature — retain in all units
- **Scenarios:** Currently over-reliant on applicant submissions, guideline revisions, and terminology inconsistencies. Diversify across modules during drafting
- **Activity codes:** Use the A1–H3 coding system from the Master Activity Menu when discussing or planning unit structures
- **Markdown conventions:** H1 for book-level titles, H2 for module headings, H3 for unit headings, H4 for section/activity headings
