## ITEM 1 — B7 for Unit 21 (new activity)

**File:** `aw-adv_mod6_revised.md`
**Unit:** Unit 21 — Drafting Consolidated Administrative Reports

**Where to insert:**
In Section B (Model Text), after the existing `#### Compare the Versions (B1)` and `#### Structural Deconstruction (B3)` activities, add the following as a third activity in the B section.

The insertion point is after the closing `---` that follows the Structural Deconstruction activity (after the line *"Discuss: Why does a predictable report structure help readers process information more efficiently?"*), and before the `### C. Language Focus` heading.

---

### Content to insert:

```markdown
#### B7. Data Interpretation Input

The table below shows status update data submitted by five offices following the distribution of Circular 2024/09.

| Office | Briefing completed | Updated form in use | Outstanding queries submitted | Implementation status |
|---|---|---|---|---|
| Office A | Yes | Yes | 2 | On track |
| Office B | Yes | No | 4 | Delayed |
| Office C | No | No | 7 | Not started |
| Office D | Yes | Yes | 0 | On track |
| Office E | Yes | Yes | 1 | On track |

**Writing task:** Using the data above, write the *Issues Identified* and *Outstanding Questions* sections of a consolidated administrative report. Your writing should:

- group offices by implementation status (do not list them one by one)
- identify the main concern(s) and how widespread they are
- summarise the outstanding queries as a pattern, not a count
- use the five-section report format from Section B as your structural guide

Write in formal, complete sentences. Aim for 60–80 words across both sections combined.
```

### Content to insert:

```markdown
**Writing task:** Using only the information in the extract above, write a plain-language version of this policy requirement for a general staff audience. Your version should:

- state what is changing and when
- explain why the change is being made
- describe what staff in affected offices are required to do
- avoid all technical references (circular numbers, article codes) unless you explain them in plain terms

Aim for 60–80 words. You will compare your version with the model text in the next activity.
```

## Notes for implementation

**On Unit 21 B7 data:** The figures in the table are designed to be pedagogically clear — three offices on track, one delayed, one not started — so that learners can identify a meaningful pattern rather than just transcribe numbers. The task explicitly discourages a list-by-list approach, reinforcing the synthesis skill that is the unit's core focus.

**On Unit 19 B6 task:** The instruction to compare your version with the model text in the next activity (B1) creates a deliberate reading-to-write loop: learners attempt the task first, then see a professional version, which deepens their engagement with the B1 comparison. This is the standard B6 pedagogical sequence (attempt → compare → reflect) and brings Unit 19's B6 into full alignment with how the activity type operates in Units 15, 18, and 22.

**On B7 placement in Unit 21:** The structural profile for Unit 21 does not currently list B7. Adding it makes the B section three activities (B1 + B3 + B7), which is on the upper end but justified here — Unit 21 is a consolidation unit in Module 6 where heavier input work supports the extended output tasks. The B7 table is short and the writing task is tightly scoped, so the activity is not burdensome relative to the unit's level.
