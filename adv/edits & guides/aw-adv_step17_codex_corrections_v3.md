### HEADING FORMAT ‚Äî Standard

The correct format for all activity headings throughout the book is established in the Unit Structural Profiles document:

> *"Each unit block lists the activity assigned to each of the 8 instructional sections, with the **activity name followed by its code in brackets**."*

The correct format is therefore: `#### Activity Name (Code)` ‚Äî for example, `#### Problem Identification (A2)`.

Module 6 n9 largely follows this standard. Modules 4 and 5 n9 use the **inverted** format throughout ‚Äî `#### Code ‚Äî Name` (e.g. `#### A4 ‚Äî Prediction Task`, `#### G2 ‚Äî Editing Workshop`) ‚Äî and must be corrected across all activity headings.

Additionally, some Module 6 headings are missing their code entirely, and Module 4 contains one non-existent code (G4) that must be replaced with the correct code.
