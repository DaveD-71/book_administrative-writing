# Unit Structural Profiles — Master Planning Table
## *Administrative Writing, Advanced*

**How to read this document**
Units are presented individually, grouped by module. Each unit block lists the activity assigned to each of the 8 instructional sections, with the activity name followed by its code in brackets. ★ marks the first appearance of a new activity type across the book. Learning goals are drafting targets for planning purposes, not final published objectives. Word count targets follow the agreed progression: ~150w (Module 1) → ~200w (Modules 2–3) → ~250w (Modules 4–5) → ~300–350w (Module 6 / Units 22–23).

---

## Typical Unit Structure

Each unit is organised into **8 sections**. Most sections contain one or two activities — so while the average unit has 12 activity codes, these map to 8 distinct stages of instruction.

| Section | Typical activity count | Example (Unit 1) |
|---------|----------------------|------------------|
| A. Opening & Activation | 1 | Problem Identification (A2) |
| B. Model Text | 2 | Compare Two Versions (B1) · Noticing Task (B2) |
| C. Language Focus | 2 | Classification / Matching (C2) · Sentence Rewriting (C3) |
| D. Guided Writing | 2 | Model Rewriting (D1) · Scenario Writing (D2) |
| E. Freer Writing | 1 | Scenario-Based Free Write (E1) |
| F. Review / Self-Edit | 1–2 | Self-Editing Checklist (F3) |
| G. Editing | 1 | Editing Exercise (G1) |
| H. Homework & Extension | 1–2 | Extension Task (H1) · Homework Writing Task (H3) |

Unit 23 (Capstone) is a deliberate exception — one activity per section, reflecting its assessment rather than instructional function.
