## P5 — Replace Sentence-Count Targets with Word Counts

**Scope:** All Freer Writing (Section E) and Homework (Section H) tasks.

**Audit result:** ✅ No changes required.

All Freer Writing and Homework tasks across Modules 1–6 already use word count targets. The conversion was completed during earlier drafting steps.

**Note:** Three instances of sentence-count wording were found in Guided Writing (Section D) tasks — specifically in D2 activities in Units 4, 6, and 7 (Module 2). These read "write a structured email of 5–7 sentences", "write a diplomatic error-notification message of 5–7 sentences", and "write a structured multi-part request of 6–8 sentences". These are **not in scope for P5**, which applies only to Freer Writing and Homework tasks. D2 (Guided Writing) tasks are shorter, more constrained exercises where sentence counts are an appropriate scaffold. No action needed.

### Edit 1 — Module 3, Unit 9: C7 Expansion Task (3 items → 4 items)

**File:** `admin-writing-adv_mod3_revised.md`

**Location:** Unit 9 (Change Management Writing) → Language Focus → C7. Expansion Task

**Type of edit:** ADDITION — add one new item (item 4) to the existing list.

**Current text (full practice activity):**

```
#### C7. Expansion Task

Each sentence below is a bare statement of fact. Expand each one into 2–3 sentences by adding the missing elements indicated in brackets.

1. *The submission deadline has changed.* [Add: effective date + reason for change]
2. *Staff should use the updated form.* [Add: what the form replaces + where to find it]
3. *The previous process is no longer valid.* [Add: what the new process involves + what staff should do now]
```

**Replace with:**

```
#### C7. Expansion Task

Each sentence below is a bare statement of fact. Expand each one into 2–3 sentences by adding the missing elements indicated in brackets.

1. *The submission deadline has changed.* [Add: effective date + reason for change]
2. *Staff should use the updated form.* [Add: what the form replaces + where to find it]
3. *The previous process is no longer valid.* [Add: what the new process involves + what staff should do now]
4. *The coordinating office has revised its communication procedure.* [Add: what specifically has changed + the benefit this brings to partner offices]
```
